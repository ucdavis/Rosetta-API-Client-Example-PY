## Rosetta API Client Example

A quick repo to demostrate how to access the UCD Rosetta API. 

Example of .env file

```bash
# Rosetta Client ID
ROSETTA_CLIENT_ID=XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

# Rosetta Client Secret
ROSETTA_CLIENT_SECRET=XXXXXXXXXXXXXXXXXXXXXXXXXXXXX

# Rosetta Base URL
ROSETTA_BASE_URL=https://xxxxxxxxxxxxxxxx.xxxxxxxxxxxxxxxxxxxxxxxx.io/api/v1/

# Rosetta OAuth URL
ROSETTA_OAUTH_URL=https://xxxxxxxxxxxxxxxxxxx.xxxxxxxxxxx.xxxxxxxxxx.io/token

# Rosetta Test IAM ID
ROSETTA_TEST_ID=XXXXXXXXXXXXXXXXXXXXXXXX

```
